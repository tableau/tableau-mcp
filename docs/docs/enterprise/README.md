# Enterprise Deployment

Tableau MCP can be self-hosted on enterprise infrastructure you control by deploying and operating
the server process and exposing it to your users' MCP clients at a URL you own. Users will authorize
via Tableau sign-in before they can connect, so you can offer broad access under your security and
identity policies. The server is highly configurable and employs a plugin-based model so operations
like telemetry and monitoring can be provided based on your needs.

## Tableau Server

Tableau Server customers **on 2025.3 or newer** should use the
[deployment guide for Tableau Server customers](./tableau-server.md).

## Tableau Cloud

As part of the 2026.2 release, Tableau MCP is rolling out a cloud-hosted service, available to all
editions. Tableau Cloud customers will be able to self-host Tableau MCP at this time, at which a
deployment guide will also be provided here.
