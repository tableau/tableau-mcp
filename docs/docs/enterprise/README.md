# Enterprise Deployment

Tableau MCP can be self-hosted on enterprise infrastructure you control by deploying and operating
the server process and exposing it to your users' MCP clients at a URL you own. With OAuth enabled,
users will authorize via Tableau sign-in before they can connect, so you can offer broad access
under your security and identity policies. The server is highly configurable and employs a
plugin-based model so operations like telemetry and monitoring can be provided based on your needs.

## Tableau Server

Tableau Server customers should use the
[Tableau MCP deployment guide for Tableau Server customers](./tableau-server.md).

## Tableau Cloud

As part of the 2026.2 release, Tableau is planning to roll out a cloud-hosted service, available to
all editions. The release will unlock the ability for Tableau Cloud customers to self-host Tableau
MCP at scale using OAuth. A Tableau MCP deployment guide for Tableau Cloud customers will also be
made available with the release.
